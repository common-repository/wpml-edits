Edit language code and name in WPML

== Description ==

Allow in Wordpress Admin area to change language code and name for WPML plugin

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wpml-edits.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In admin area a new option appears under Settings: WPML Edit
1. Need to have WPML plugin installed and activated
